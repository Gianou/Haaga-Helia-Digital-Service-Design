Test

User evaluation evaluation was conducted for the low-fidelity prototype that was created for the concept. During the evaluation, the participant was given six tasks and the user was instructed to think out loud their thought process.

During the evaluation, we tried to interact with the evaluator as little as possible, to promote genuine reactions.

Tool used for this stage: Canva Computer, screen, webcam

In order to test our prototype, we defined six test scenarios. The goal was to see if the app is intuitive and easy to use. Sara provided the template and all members defined at least one scenario.
